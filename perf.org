* Linux perf
** Trace an application

   #+BEGIN_SRC shell
     perf record --call-graph lbr ./test
   #+END_SRC

** Flat mode (pur sampling) no call-stack

   #+BEGIN_SRC shell
     perf record ./test
   #+END_SRC
  
  (better callstack for hotspot)

** Output
*** Base output

    #+BEGIN_SRC shell
      perf report --stdio
    #+END_SRC
  
*** Curses output

    #+BEGIN_SRC shell
      perf report
    #+END_SRC
  
** Annotate only one function

    #+BEGIN_SRC shell
      perf annotate main [--stdio]
    #+END_SRC
 
** Wrapper for MPI

   #+BEGIN_SRC shell
     #!/bin/sh

     H=`hostname`
     P=$$

     perf record -o perf-${H}-${P}.data  $@
   #+END_SRC
  
*** Usage

    #+BEGIN_SRC shell
      mpirun -np 4 mpiperf ./program options
    #+END_SRC

** Profile ALL the system (in root or with good parameter)

   #+BEGIN_SRC shell
     perf record -a --call-graph lbr
   #+END_SRC

*** With curses

    #+BEGIN_SRC shell
      perf top
    #+END_SRC
