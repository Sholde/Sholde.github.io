1 Introduction

valgrind is a compilation of many tools. Here we will just use Memcheck module which are the default.

2 Usage

Compile your program and then run it with valgrind. 

$ gcc prog.c -o prog
$ valgrind ./prog

3 Example

3.1 Without leak

int main()
{
  int a[5];
  return 0;
}
